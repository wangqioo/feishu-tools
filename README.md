# 飞书表格函数面板

跨飞书表格的可视化函数操作工具。无需写公式，通过图形界面配置 VLOOKUP / XLOOKUP / INDEX-MATCH / SUMIF / COUNTIF / SUMIFS，支持在线飞书表格与本地 Excel/CSV 文件作为数据源，结果可直接写回飞书或导出 Excel。

## 功能概览

### 数据源管理

- **在线飞书表格**：粘贴表格链接自动提取 Token，一键拉取所有工作表列表，多选批量导入
  - 支持企业内网代理和飞书开放平台 API 两种接入方式
  - 自动分页拉取全部行，无行数限制
  - 隐藏工作表自动标注 `[隐藏]`，默认不选中
  - 注意：飞书 API 无法识别隐藏行/列，如有隐藏内容会一并导入
- **本地文件**：支持 `.xlsx` `.xls` `.csv` `.tsv`
  - 自动以文件名填充数据源名称
  - xlsx 支持多工作表多选批量导入
  - 自动过滤隐藏行和隐藏列，并告知跳过数量
  - 自动探测 CSV 编码（UTF-8/GBK/Latin-1）
- 数据缓存到本地 SQLite，断网后仍可进行函数计算
- 同步时如遇权限不足，给出详细原因说明和操作建议

### 函数构建器（可视化配置）

| 函数 | 说明 |
|------|------|
| **VLOOKUP** | 垂直查找，支持一次返回多列 |
| **XLOOKUP** | 搜索列与返回列可来自不同数据源（跨表） |
| **INDEX/MATCH** | MATCH 表和 INDEX 表完全独立，最灵活的跨源查找 |
| **SUMIF** | 条件求和，展示匹配明细 |
| **COUNTIF** | 条件计数，展示匹配行 |
| **SUMIFS** | 最多 4 个条件的组合求和 |

所有函数均支持 5 种匹配模式：

| 模式 | 说明 |
|------|------|
| 精确匹配 | 完全相同（大小写不敏感） |
| 包含匹配 | 单元格内容包含关键字 |
| 前缀匹配 | 以关键字开头 |
| 后缀匹配 | 以关键字结尾 |
| 正则匹配 | 使用正则表达式 |

### 结果预览与导出

- 实时搜索过滤结果
- 匹配行绿色高亮，未匹配行红色标记
- 导出 Excel（保留颜色标记）
- 复制为 TSV 粘贴到任意表格软件
- **写回飞书**：选择目标数据源、起始行/列、写入哪些结果列，支持跳过未匹配行

### 在线编辑器

- 将飞书表格数据加载为可编辑网格
- **双击任意单元格**进行编辑，Enter/Tab 确认，Esc 取消
- 修改过的行自动高亮（黄色），顶部显示未保存数量
- 「保存修改到飞书」批量写回所有改动
- 「粘贴剪贴板列」：将剪贴板中的换行文本一键填入指定列（常用于把函数结果填到目标表）

## 快速开始

```bash
python feishu_vlookup_panel.py
```

无需提前安装依赖，首次运行自动安装 `requests` 和 `openpyxl`。

## 典型使用流程

```
① 数据源管理
  → 添加数据源（在线飞书 或 本地 Excel/CSV）
  → 粘贴飞书链接，自动提取 Token
  → 点击「拉取工作表列表」，勾选需要的工作表
  → 点击「添加并导入」，实时显示每个工作表拉取进度

② 函数构建器
  → 选择函数类型（如 VLOOKUP）
  → 配置查找值来源、查找范围、返回列、匹配模式
  → 点击「执行函数」

③ 执行结果
  → 查看匹配/未匹配统计
  → 搜索过滤
  → 「写回飞书」直接填入目标表格列
  → 或「导出 Excel」

④ 在线编辑（可选）
  → 直接在工具内修改飞书表格内容
  → 保存修改到飞书
```

## API 接入说明

本工具支持两种接入飞书的方式，在「API 设置」页配置：

| 方式 | 适用场景 | 需要配置 |
|------|----------|----------|
| 企业内网代理（默认） | 连接公司内网时 | 内网代理地址、App ID、User ID（工号） |
| 飞书开放平台 API | 任意网络环境 | user_access_token |

### Token 获取方式

打开飞书表格，复制浏览器地址栏中的链接，直接粘贴到 Token 输入框，工具会自动提取。

例如：`https://xxx.feishu.cn/sheets/shthqz7lKPJt9UGF4FOIU1uyTUh?sheet=1UuGvI`
自动提取为：`shthqz7lKPJt9UGF4FOIU1uyTUh`

### 权限不足时的常见原因

1. 当前账号未被添加为表格「可查看」成员 → 联系表格所有者授权
2. userId（工号）填写有误 → 检查 API 设置页
3. Access Token 已过期 → 重新生成并填入
4. 表格所在空间限制了组织外访问 → 联系空间管理员

## 数据说明

- 所有数据缓存于 `feishu_vlookup_data/cache.db`（SQLite），不上传任何数据
- 本地文件数据源以文件路径记录，「同步」时重新读取文件
- 在线数据源每次「同步」才更新缓存，函数计算使用的是缓存快照
